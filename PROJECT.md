# Dolcove

Dolcove is a private-group social app for close friend groups and small communities.

The MVP focuses on one core loop:

group conversation -> memory -> planning -> recap -> more group activity

Dolcove is not a public social feed. It is designed for private circles where people already know each other and want a better way to stay connected, make plans, and preserve shared moments.

## MVP goals

The MVP must allow users to:
- sign up and log in
- create a private group
- view their groups
- exchange messages inside a group
- create and respond to plans
- view memory highlights derived from group activity
- receive recap content summarizing activity

## Product principles
- Private by default
- Small-group focused
- Useful before clever
- Memory should feel helpful, not creepy
- Planning should reduce friction, not add ceremony
- Recaps should feel short, warm, and shareable
- The app should feel lightweight and socially natural

## Non-goals for MVP
- Public discovery feed
- Marketplace or creator economy
- Complex profile systems
- Heavy recommendation graph
- Full monetization system
- Enterprise/work use cases

## Core entities
- User
- Group
- Group membership
- Message
- Plan
- Memory
- Recap
- Notification

## Success criteria for MVP
- A user can create a group and view it on mobile
- A group can use chat reliably
- A group can create a plan with minimal friction
- Memory highlights feel relevant and useful
- Recap content reflects real group activity
- Core flows are stable on mobile
